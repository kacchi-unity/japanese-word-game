## japanese-word-game
# 프로젝트 소개
Unity를 이용해 일본어를 최대한 쉽고 재미있게 배울 수 있도록 만든 단어 학습 미니 게임입니다.
한글 뜻을 입력하면 일본어 한자를 들고있는 적에게 공격을 가하고 점수를 얻는 게임을 구현 할 예정입니다.

# 제작 목적
-Unity 기초 쌓기
-미니 프로젝트 제작 경험 만들기

# 사용 기술
-Unity
-Unity에 내재된 C#
-폰트
-일본어 한자 외부 데이터 관리

# 개발 기록
0 프로젝트 구조 설계 및 기획
1 유니티에서 일본어 한자 지원을 위해 외부 일본어 한자 지원 폰트를 가져와 메인폰트로 구성했다.
1 외부에서 가져온 메인 폰트에서 지원하지 않는 문자 깨짐을 해결하기 위해 Unity의 TextMeshPro의 Fallback Font 기능을 사용하여 또 다른 폰트를 서브 폰트를 활용해 보완했다.
2 화면에 표시된 일본어 단어(食べる)를 보고 사용자가 입력창에 한글 뜻을 입력하면, 버튼 클릭을 통해 정답 여부를 콘솔에 출력하는 기능을 구현하였다.
